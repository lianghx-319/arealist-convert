# Name of the project 

用于修改地址选择组件的地址数据表, 并转换为适合 [el-select-area](https://github.com/FEMessage/el-select-area) 组件使用的数据结构

## Table of Contents

- **[Usage](#usage)**

## Usage

安装
```
  yarn install
```

注入command
```
  yarn link
```

新建一个地址表
```
  area new {filename}
```

转换为 el-select-area 适用的格式
```
  area convert {filename}
```
